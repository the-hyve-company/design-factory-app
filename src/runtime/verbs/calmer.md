You are calming an existing design that feels overstimulating, busy, or shouty. Reduce visual noise without removing function.

Method

1. **Tone down the palette** — collapse near-duplicate colors, reduce chroma on accents, tint everything slightly toward a single neutral hue. Cap saturated surface area at ≤10% of the viewport.
2. **Reduce shadows** — pick one elevation level, apply consistently. Remove decorative shadows. No stacked layers of differently-sized shadows.
3. **Pull type weight back** — drop heading weights from 700-900 to 600-700. Slightly reduce display sizes.
4. **Quiet the motion** — reduce entrance distances, soften eases, shorten durations. Or remove motion that doesn't serve comprehension.
5. **Spacing breath** — increase whitespace between unrelated sections. Looser, not tighter. The eye needs places to rest.
6. **Border restraint** — 1px borders maximum, lower opacity. No double-borders, no thick accent borders.
7. **Decorative cleanup** — remove pill chips, badges, decorative icons that aren't carrying meaning. Each remaining element should earn its place.

What NOT to do

- Strip personality entirely — calm ≠ generic. The brand voice stays.
- Remove functional elements (CTAs, forms, navigation) — those keep their prominence.
- Switch to monochrome by default — only if the design already has a color identity worth keeping, do that.

Test
The design should feel composed, like the noise turned down. Information hierarchy should be clearer, not the same.

Output

- The full modified HTML document. <!DOCTYPE html> first.
- Code only. No prose. No markdown fences.
